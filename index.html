부동산거래 사이트

임차인과 임대인, 공인중개사 부동산 거래 주체들을 유기적으로 잇는 통합 주거 웹사이트 구축 하기 위한 웹페이지 구현하였습니다.
  • 개발환경 :
    - 운영체제 : Window OS
    - 개발도구 : Spring Framework, Visual Studio Code
    - DBMS : Mysql
    - Server : Apache Tomcat 9.0
    - Language : Java 11, JavaScript
    - 디자인툴 : Bootstrap

  • 프로젝트 시 활용해본 대표 기술:
    - Java, Spring Framework / Mybatis / JSP / JQuery / Ajax / JavaScript

  • 구현 기능 :
    - 메인페이지 : 공지사항, 지도검색, 부동산 뉴스, 채팅, 반응형 뷰

    - 로그인 : 일반고객 / 사업자 로그인 및 회원가입, 카카오 소셜 로그인, 네이버 메일 API, 아이디 찾기, 비밀번호 찾기, 회원탈퇴

    - 지도페이지 : 전체 조회, 마커 클릭 이벤트, 법정동 검색 후 조회, 상담 가능 매물 조회, 상세페이지로 정보 전달하며 이동

    - 분양 페이지 : 청약 정보 조회, 분양 가이드, 청약가점계산기, 분양 일정 조회

    - 알림 페이지 : 운영 알림, 관심 분양 알림

    - 일반회원 마이페이지 : 최근 본 방 목록, 내 매물 등록, 내 매물 조회/삭제, 찜한 목로, 내 정보 수정하기, 쿠폰 목록

    - 공인중개사 회원 마이 페이지 : 등록한 매물, 매물신청 리스트, 상담신청 리스트, 내 정보

    - 매물 페이지 : 매물상세페이지, 매물 삭제, 매물 상담 신청, 찜하기, 허위매물 신고, 매물 내 게시글 작성, 매물 내 게시글 삭제, 게시글에 댓글 등록/신고/삭제

  • 담당 역할 :
    - 로그인 페이지(일반회원, 중개사 회원가입 및 로그인/ 아이디 찾기, 비밀번호 찾기 / 카카오 소셜로그인, 네이버 메일 API), 마이페이지(일반회원, 중개사 회원 정보수정/ 회원탈퇴)

  • 회고 :
    - 프로젝트 진행시 처음 다뤄보는 내용들이 많아 어려움이 있었지만 팀원들과의 원활한 소통과 협업을 통해 잘 해결해 나갔으며, 효율적인 일정관리를 통해 빠르게 작업을 마칠 수 있었습니다. 기술적인 부분에서 Ajax를 이용한 비동기 처리 및 API에 대해 한층 더 알 수 있었습니다.
