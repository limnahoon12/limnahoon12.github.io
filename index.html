  • 개발환경 :
    - 운영체제 : Window OS
    - 개발도구 : Spring Framework, Visual Studio Code
    - DBMS : MySQL
    - Server : Apache Tomcat 9.0
    - Language : Java 11, JavaScript, jQuery, HTML5, CSS
    - 디자인툴 : Bootstrap
  • 프로젝트 시 활용해본 대표 기술:
    - Java / Spring Framework / Mybatis / JSP / JQuery / Ajax / HTML5 / CSS / Bootstrap
  • 구현 기능 :
    ◇페이지 내 회원가입, 카카오 API, 네이버 메일 API를 통한 회원가입
    ◇부동산 매물 API를 가공하여 지도에 마커로 표시, 실거래가 표현, 지도 검색 기능
    ◇공인중개사가 매물 등록/삭제, 등록된 매물 안에 게시판과 댓글
    ◇청약 정보와 분양 일정
    ◇실시간 알림 기능
    ◇관리자와 1:1 문의 채팅

    ◇마이페이지
    ▶공인중개사 경우
    - 등록한 매물 조회 및 매물 등록
    - 회원이 신청한 매물 신청 리스트 조회 수락/거절
    - 회원이 신청한 상담 리스트 조회 및 수락/거절
    - 나의 정보 조회 및 수정

    ▶일반 회원일 경우
    - 최근 본 방 5개 조회
    - 내가 등록한 매물 조회
    - 찜한 매물 조회
    - 나의 정보 조회 및 수정
    - 쿠폰함에서 쿠폰 조회 및 사용

    ◇관리자 페이지
    - 회원 조회
    - 회원가입한 공인중개사 승인
    - 공지사항 등록/수정/삭제
    - 회원별 신고 관리
    - 쿠폰 등록 및 발급
    - 채팅 대응 및 채팅방 삭제
    - 등록된 매물 조회, 검색, 삭제

    - 공인 중개사 회원 프로필 이미지 변경
    - 일반회원, 중개사 회원 탈퇴시 비밀번호 인증 후 일치 할 경우 탈퇴 진행행
